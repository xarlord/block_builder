# Architect AI - Block Builder

An intelligent design tool that transforms natural language prompts into structured compositions using Magnatile-like components.

## Project Status

🚧 **Early Development** - Specification phase

## Overview

Architect AI uses a curated library of five core components to build designs from natural language descriptions. The AI acts as a "Digital Architect," calculating spatial coordinates and rotations to assemble complex objects.

## Core Components

- Solid Square (3x3)
- Window Square (3x3)
- Equilateral Triangle
- Right Triangle (90°)
- Isosceles Triangle

## Tech Stack (Planned)

- **Frontend**: Android / Kotlin with Jetpack Compose
- **AI**: LLM integration with constraint validation
- **Rendering**: SVG-based component system
